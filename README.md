## 🚀 Proje Hakkında

Bu proje, kullanıcıların harita üzerinde konum seçerek rota oluşturmasını sağlayan dinamik bir web uygulamasıdır. Harita üzerinde işaretleyiciler (markers) eklenir ve kullanıcı etkileşimlerine göre güncellenir.

Modern frontend geliştirme teknikleri kullanılarak geliştirilmiştir.

---

## ✨ Özellikler

- 📍 Harita üzerinde konum seçme
- 🧭 Başlangıç ve varış noktası belirleme
- ➕ Dinamik marker ekleme
- 🗺️ İnteraktif harita kullanımı
- 🔄 Gerçek zamanlı UI güncellemeleri
- 📱 Responsive tasarım

---

## 🛠️ Kullanılan Teknolojiler

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Harita API (Leaflet / Mapbox vb.)
- GeoJSON / koordinat sistemleri

---

## 📚 Bu Projede Neler Öğrendim

### 🔹 JavaScript

- Event-driven programming
- Dinamik veri yönetimi
- Modüler kod yazımı

### 🔹 Harita Teknolojileri

- Marker (işaretleyici) kullanımı
- Koordinat sistemi (latitude / longitude)
- Harita eventleri (click, zoom, move)

### 🔹 UI / UX

- Kullanıcı etkileşimine göre arayüz güncelleme
- Sidebar ve harita senkronizasyonu

---

## ⚙️ Uygulama Mantığı

1. Kullanıcı harita üzerinde bir noktaya tıklar
2. Seçilen koordinatlar alınır
3. Haritaya marker eklenir
4. Sidebar’da rota bilgisi güncellenir
5. Başlangıç ve varış noktaları belirlenir
